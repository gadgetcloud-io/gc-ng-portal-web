## Usage Tips

### When to Show Upgrade Prompts

**DO**:
- Show when user is 80% toward a limit (proactive)
- Show when user hits a limit (reactive)
- Show during onboarding (for users who add 5+ devices immediately)
- Show after a successful action (e.g., "Great! You added a device. Want to add more? Upgrade for 10 devices!")

**DON'T**:
- Show too frequently (max once per day)
- Show during critical tasks (e.g., don't interrupt file upload)
- Show immediately after user dismisses a prompt (wait 7 days)
- Show to users who downgraded recently (wait 30 days)
